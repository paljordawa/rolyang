 Rolyang Music Album

What's included:
- Modern landing, library, Album details, and player pages
- Player with queue preview, sleep timer, speed, mediaSession integration
- Bottom navigation for small screens and responsive layouts
- Illustration/cover copied from uploaded image (public/illustration.png)

Run:
1. npm install
2. npm run dev
Place audio files under public/audio/ matching src/data/data.json paths.
